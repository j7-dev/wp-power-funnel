# 報名管理頁面

## 描述

管理員在 WordPress 後台檢視與管理所有 pf_registration 報名紀錄的列表頁面。
此頁面使用 WordPress 原生 CPT 列表介面（edit.php?post_type=pf_registration），
管理員可透過此頁面審核報名、變更報名狀態。

## 關鍵屬性

- 使用 WordPress CPT 原生列表介面
- 顯示欄位：報名標題、活動 ID、用戶 ID、身份提供商、推廣連結 ID、報名狀態
- 狀態篩選：pending / success / rejected / failed / cancelled
- 操作：變更報名狀態（透過 WordPress 內建的 post status 切換）

## 互動流程

1. 管理員進入 WordPress 後台 > pf_registration 列表
2. 查看所有報名紀錄
3. 點擊某筆報名紀錄進入編輯頁面
4. 變更 post_status（如從 pending 改為 success 或 rejected）
5. 狀態變更觸發 transition_post_status hook
6. 系統觸發 power_funnel/registration/{status} action
7. 對應的 LINE 通知邏輯被執行
